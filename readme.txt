== Description ==

[Conditional Payment Methods for WooCommerce](http://www.wpexpertdeveloper.com/woocommerce-conditional-payment-methods/) provides features conditionally display payment gateways on checkout based on wide range of conditions. This plugin supports user types based, user shipping country based and WooCommerce product type based payment methods in checkout.
 
> **Practical Usage and Benifits**
>
> * Enable Paypal payment gateway (Any payment method) only for guests.
> * Enable Paypal payment gateway (Any payment method) only for members.
> * Enable Paypal payment gateway (Any payment method) only users from Australia and United Status
> * Enable Paypal payment gateway (Any payment method) only for users who purchase WooCommerce simple products
> * Enable Paypal payment gateway (Any payment method) only for members from Australia
> * Enable Paypal payment gateway (Any payment method) only for guests from United Status or France
> * Enable Paypal payment gateway (Any payment method) only for members from United Kingdom who purchases only simple products.
> * Enable Paypal payment gateway (Any payment method) only for guests from United Status or France who purchases custom product type created by another plugin.

> **Features**
>
> * [Enable/ Disable any payment method on WooCommerce checkout based on guests or logged in members](http://www.wpexpertdeveloper.com/woocommerce-conditional-payment-methods/)
> * [Enable/ Disable any payment method on WooCommerce checkout based on users shipping countries](http://www.wpexpertdeveloper.com/woocommerce-conditional-payment-methods/)
> * [Enable/ Disable any payment method on WooCommerce checkout based on WooCommerce product type](http://www.wpexpertdeveloper.com/woocommerce-conditional-payment-methods/)
> * [Supports single set of conditions for each payment method](http://www.wpexpertdeveloper.com/woocommerce-conditional-payment-methods/)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-conditional-payment-methods` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Follow the <a href="http://www.wpexpertdeveloper.com/woocommerce-conditional-payment-methods/">Documentation and Usage</a>.
